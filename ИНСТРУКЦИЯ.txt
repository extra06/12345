ЧТО НУЖНО СДЕЛАТЬ:
────────────────────────────────────────────────────────────

1. Открой папку: static-site

2. Выдели ВСЕ файлы (Ctrl+A):
   - index.html
   - styles.css
   - script.js
   - photo1.png - photo10.png

3. Скопируй (Ctrl+C)

4. Зайди на GitHub:
   https://github.com/extra06/apartament-site

5. УДАЛИ ВСЕ СТАРЫЕ ФАЙЛЫ:
   - index.html
   - assets/ (папка)
   - photo*.png
   - README.md

6. Загрузи НОВЫЕ файлы:
   - Нажми "Add file" → "Upload files"
   - Перетащи ВСЕ файлы из static-site
   - Или просто вставь (Ctrl+V) в браузер
   - Commit changes

7. Настрой GitHub Pages:
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
   - Save

8. Подожди 1-2 минуты

9. Открой сайт:
   https://extra06.github.io/apartament-site/
